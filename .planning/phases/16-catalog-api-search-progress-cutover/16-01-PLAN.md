<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| catalog query helpers -> public API DTOs | catalog/internal rows cross into user-visible responses here |
| search/filter inputs -> catalog search document queries | untrusted query strings drive database filtering |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-16-01 | I | `GetItemDetail` / `ListSeriesSeasons` | mitigate | build responses only through `BuildCatalog*` helpers so source evidence and field states remain scalar-safe and raw JSON blobs never cross the API boundary |
| T-16-02 | T | `ListItems` query filters | mitigate | constrain top-level browse to catalog root rows and use typed filter inputs so untrusted query strings cannot coerce handlers back into legacy episode-group behavior |
</threat_model>
