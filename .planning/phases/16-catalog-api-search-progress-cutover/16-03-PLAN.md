|----------|-------------|
| catalog progress request -> user_item_data | user-controlled playback state becomes durable per-user data |
| item/asset identifiers -> asset_items validation | untrusted IDs are used to target an item/asset relationship |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-16-05 | T | `UpdateCatalogProgress` | mitigate | require catalog item existence and validate `asset_id` through `asset_items` before touching `user_item_data` |
| T-16-06 | R | completion/play-count updates | mitigate | increment `play_count` only on incomplete -> complete transitions and preserve timestamps so repeated writes do not fabricate extra plays |
