| T-12-05 | T | migration timestamp parsing | mitigate | Accept only RFC3339 timestamps and reject malformed values with 400 instead of coercing them. |
| T-12-06 | I | `/api/v1/system/info` migration block | accept | Expose only the three non-secret migration flags; do not surface unrelated settings rows or secret values. |
